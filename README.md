## How to use

```
yarn install
```
Run mock server
`yarn json:server`
Then run the project
`yarn dev`

First version features:

  - Main page
  - Main structure
  - Thread list
  - Profile page
  - Archived threads page
  - Contacts page
  - Implementation of search by contact name
  - Messages of thread page
  - Implementation of sending a new message (improvement needed)
